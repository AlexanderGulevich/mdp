# TMUX
__---  Ctrl + b ? --- shortcuts__


### Sessions

Commands |  Actions
--|--
tmux | start session
tmux kill-session -t mysession  | kill/delete session
tmux kill-session -a  | kill ses but the current
Ctrl + b $ | Rename session
Ctrl + b d | Detach from session
Ctrl + b s  | Show all sessions
Ctrl + b (  )  | Move to next session

### Windows

Commands |  Actions
--|--
Ctrl + b c | Create window
Ctrl + b & | Close
Ctrl + b p | Previous
Ctrl + b n | Next 
Ctrl + b  0-9| Switch

### Panes

Commands |  Actions
--|--
Ctrl + b "   Ctrl + b % | Split
Ctrl + b {    } | Move
Ctrl + b     -> <- | Switch 
Ctrl + b q | Number
Ctrl + b z | Pane zoom
Ctrl + b ! | Conwert to window 
Ctrl + b  Ctrl + <-- --> | Resize 
Ctrl + b x | Close

### Copy mode   (LIKE VIM)	

Commands |  Actions
--|--
Ctrl + b [ | Start
Ctrl + b PgUp |  Scroll
q | Quite
g/G | Go to top/bottom line
